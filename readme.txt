== Description ==

Provides a widget to display the last or next match for a specified team. Currently supporting German Bundesliga 1-3. Powered by openligadb.de

== Installation ==

1. Upload the whole plugin-folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to "Widgets" and put the soccr-widget to your sidebar
